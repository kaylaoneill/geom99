# Workforce

## Working with Workforce
- Go to Arcgis online and go to workforce through the menu
- Click "new project"
- Add assignment types:
- <img width="727" alt="Screenshot 2024-03-21 204150" src="https://github.com/kaylaoneill/geom99/assets/146447016/7a14c1dc-940b-4d07-bc6d-75977caa0d0a">
- Add users
  - Can look through list in organization
  - Once added, you can add contact details (a cell phone number) to the account
  - Once complete, you will get this screen:
  - <img width="704" alt="image" src="https://github.com/kaylaoneill/geom99/assets/146447016/dfec13e3-c9c5-4a7b-aba0-fb76e932b25a">
- Click open project
- <img width="845" alt="image" src="https://github.com/kaylaoneill/geom99/assets/146447016/de5d5ab1-cb41-497e-a795-ae2ddd71f957">
- Add an assignment:
- Click the "Add Assignment" button
- Click a type from the list you created in set up
- Assign it to a user you added
- Fill in the rest of the assignment details, once it has everything it needs, create assignment button will become clickable
- <img width="458" alt="image" src="https://github.com/kaylaoneill/geom99/assets/146447016/0cc2812d-c07f-486f-9049-57ab4d0e46a9">
- Now you can see open assignments on the browser:
- <img width="947" alt="image" src="https://github.com/kaylaoneill/geom99/assets/146447016/a6144f56-6434-4851-a3a6-9e535b1d6e26">
